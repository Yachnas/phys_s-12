Assignment


Program a microcontroller to produce an output using at least one button for input. Include a conditional statement and a for() loop. Then think about your final project and any additional components you might want. Start working on the 3D design for your project, especially components you might want to be lasercut or 3D printed.

Circuit building


I apoligize, I could not find my buttons so instead I changed the input to the accelerometer. I gathered the necessary materials to build the circuit: an ESP32 Huzzah microcontroller, some female and male connections, a green LED, a 470 ohm resistor, and an accelerometer. I then construct the circuit, with the accelerometer acting as an input and the LED acting as an output. The accelerometer and the ESP32 microcontroller were previously wired from the last assignment, so all I had to wire was the LED portion of the circuit.



Microcontroller firmware


I then wrote the firmware on the microcontroller. I wanted the LED to blink quickly when acceleration was detected, so I wrote a for() loop that would run if the magnitude of the accelerometer was greater than 18000. Since I was using raw sensor values this number was an arbitrary value roughly equal to the magnitude when any outside force acted on the circuit.


Completed circuit


After finalizing the firmware I uploaded it to the microcontroller and the circuit worked successfully. The LED pulsed when the circuit was moved and remained off otherwise.

For my final project I want to make a simple controller with no buttons. I want to be able to control a character or an object on the screen with simple movements. For instance, maybe I could have a character perform a movement when the controller is shaken and perform another movement when the controller is moved from the left to the right. I would like the controller to be wireless and for it to connect to a desktop computer running the game when it is moved.
